**Getting Started**

    1. You need to install Node.js v16.6.0 or higher.
    2. Clone the repository https://github.com/itshyko/DiscordBot-JS-Base.git
    3. Register for a Discord bot token.

**Before starting**

Before you start the bot you'll need to add your bots token in the config.json file.

**Install modules**

    1. Launch an Admin Command Prompt in the Bot directory.
    2. Make sure Package.json is in the file location
    3. npm install
    4. node bot.js
